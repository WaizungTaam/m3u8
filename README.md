# M3U8

**M3U8** is a parser for HTTP Live Streaming (HLS) playlist `.m3u8` files.

The protocol is described in [rfc8216](https://datatracker.ietf.org/doc/html/rfc8216).


## Installing

Install with `pip`:
```console
$ pip install -e git+https://github.com/WaizungTaam/m3u8
```
